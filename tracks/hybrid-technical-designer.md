# Hybrid Technical Designer Track

## Goal

Become a product thinker who understands design, systems, AI, and light coding.

## Core Skills

- UX/UI fundamentals
- Product thinking
- System design
- API understanding
- AI logic
- Basic coding (HTML, CSS, JS, React basics)

## Weekly Outputs

- 1 product breakdown
- 1 system map
- 1 technical logic explanation
- 1 workflow improvement

## Focus Areas

- frontend vs backend understanding
- APIs and JSON
- AI capabilities and limits
- decision-making systems

## Proof Examples

- UI improvement analysis
- system architecture diagram
- API flow explanation
- AI fallback logic

## Outcome

Ability to design realistic systems and collaborate with developers effectively.
