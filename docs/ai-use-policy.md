# AI Use Policy

ProofPath OS supports AI-assisted learning, but requires responsible usage.

## Allowed Use

- summarizing concepts
- suggesting ideas
- generating drafts
- helping structure thoughts

## Required Responsibility

- verify important claims
- link real sources where possible
- mark AI-assisted content when relevant

## Not Allowed

- storing unverified AI outputs as truth
- fake citations or sources
- presenting AI hallucinations as facts

## Rule

AI assists thinking. It does not replace validation.

## Best Practice

- use AI → generate idea
- validate → confirm correctness
- apply → use in real scenario
- prove → create output
