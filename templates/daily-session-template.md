# Daily Session Template

Use this template for the daily 1-hour ProofPath OS session.

## Date

YYYY-MM-DD

## Track

Example: Hybrid Technical Designer

## Topic

What are you learning today?

## 1. Learn — 15 minutes

Write the core concept in simple language.

## 2. Apply — 15 minutes

How does this apply to your real work, product, business, portfolio, or workflow?

## 3. Prove — 25 minutes

Create a visible proof output.

Valid proof examples:

- product breakdown
- UX improvement
- system map
- decision memo
- code or logic snippet
- workflow redesign
- AI risk/fallback analysis

## 4. Journal — 5 minutes

Answer:

- What became clearer?
- What is still confusing?
- What will I improve next?

## Completion Rule

No proof = session not counted.
